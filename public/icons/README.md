# App Icons

Platzieren Sie hier Ihre App-Icons in folgenden Größen:

## Erforderliche Icons:

- `favicon.png` (32x32 oder 16x16)
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Icon-Generierung:

Sie können Icons online generieren mit Tools wie:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Empfehlungen:

- Verwenden Sie ein quadratisches Logo
- Hintergrundfarbe: #c8102e (Wölfleder Rot)
- Format: PNG mit Transparenz
- Für iOS: Icons sollten keine Transparenz haben

## Beispiel-Design:

Ein großes "W" in Weiß auf rotem (#c8102e) Hintergrund
